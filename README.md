# CRUD-Employee-Details

The project “ Employee Details Management System ” is a web application in Angular using other additional modules and JSON Server. This Project involves and covers all the major aspects of angular and also this has all the operations that are likely to be present in all web -  application. These major operations include the CRUD operations which include: create, read, update and delete. This project not only invloves the front end but also a live server and Apis that were added during the implementation of the project. This project provides reusable and easily alterable code. Any new programmer can easily understand the functionalities and components as each operation has a different set of components.This project binds all the functionalities that any major application needs and hence covers all the concepts of angular.
